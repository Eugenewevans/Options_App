## Development Note:
The game uses fallback assets (colored rectangles and emojis) when actual assets are not available, so it will run even without these files.

## Asset Guidelines:
- Images: PNG format, optimized for mobile
- Sounds: MP3 format, compressed for web
- Fonts: TTF format, web-safe
- Animations: JSON format (Lottie) or sprite sheets
